# Diabetes-Prediction-System
Diabetes is one of the most common chronic diseases worldwide, affecting millions of people and leading to severe health complications if not detected early. Early detection of diabetes is important to prevent serious complications such as heart disease, kidney failure, nerve damage, and vision loss. Deep Learning techniques, particularly Artificial Neural Networks (ANN), have shown promising results in analyzing complex healthcare data and making accurate predictions. This project uses clinical parameters such as BMI, physical activity, stroke, HighBP, HighChol, smoker etc.
Artificial Neural Network (ANN) to predict whether a person is diabetic or non-diabetic using parameters such as age, gender, Body Mass Index (BMI), hypertension status, heart disease history, smoking history, HbA1c level, and blood glucose level. This model is evaluated using performance metrics like Accuracy, Precision, Recall, F1-Score.

Problem Statement: This project aims to predict diabetes at an early stage using Artificial Neural Networks (ANN) and provide explainable insights with personalized health recommendations that analyzes health parameters such as BMI, physical activity, High BP, High Chol, Stroke etc., to classify individuals as diabetic or non-diabetic which helps in early detection of diabetes.

Dataset Used: Dataset Name: Diabetes Prediction Dataset Source: Healthcare and Lifestyle Data Total Records: 100,000 (Non – diabetic : 91,500 ; Diabetic : 8,500) Total Features: 9 Attributes Input Features: Gender, Age, Hypertension, Heart Disease, Smoking History,BMI, HbA1c Level, Blood Glucose Level Target Variable: Diabetes (0 = Non-Diabetic, 1 = Diabetic) Dataset Type: Structured Healthcare Dataset Prediction Type: Binary Classification Preprocessing Techniques: Label Encoding, Feature Scaling Purpose: Predict diabetes risk and provide personalized health recommendations
